# Real-Time Chat Application

- Built a real-time chat application using React.js, Node.js and Socket.io.
- Supports instant messaging between multiple users.
- Implemented real-time communication using WebSockets.
- Added user notifications and typing indicators.
- Designed responsive UI for desktop and mobile devices.
- Integrated frontend and backend for smooth message delivery.
- Developed using React + Vite for fast performance and better development experience.
